# Topic 1: Vectors and Matrices

Welcome to your first study session! Based on our curriculum plan, this section covers the absolute basics of linear algebra: representing data as vectors and matrices, and the foundational operations that power machine learning.

## 📖 Reading Assignment

Begin by reading the specific sections in your textbooks:

### 1. Mathematics for Machine Learning (MML)
**Read Chapter 2, Sections 2.1 to 2.3**
- **2.1 Systems of Linear Equations:** Understand how matrices represent systems of equations.
- **2.2 Matrices:** Grasp matrix addition, scalar multiplication, and matrix multiplication.
- **2.3 Solving Systems of Linear Equations:** Learn about the identity matrix, inverses, and the basic mechanics of solving $Ax = b$.

### 2. Linear Algebra and Learning from Data (Strang)
**Read Part I, Section 1.1**
- **I.1 Matrix-Vector Multiplication:** Focus deeply on Strang's perspective of seeing matrix multiplication as a *combination of columns* and a *combination of rows*. This is a crucial intuition for machine learning!

## 🎯 Goals for this Section

By the end of this reading, you should be able to:
1. Explain what a vector and a matrix are intuitively.
2. Perform vector addition, scalar multiplication, and calculate the dot product.
3. Perform matrix multiplication and understand rules like non-commutativity ($AB \neq BA$).
4. Explain what an Identity matrix and an Inverse matrix do.
5. Code these simple mathematical operations from scratch in Python (we have `exercises.py` ready for you).
