# Math

Use this folder for the foundations you need before ML theory.

## Subfolders
- `01-linear-algebra`: vectors, matrices, eigenvalues, vector spaces
- `02-calculus`: derivatives, gradients, partial derivatives, chain rule
- `03-probability`: random variables, distributions, expectation, Bayes theorem
- `04-statistics`: estimation, hypothesis testing, bias-variance tradeoff

## Suggested workflow
- Keep one note file per topic
- Add small Python exercises for each concept
- Use checkpoints before moving on to ML theory
