# Journal

Use this folder to track your learning process over time.

## Subfolders
- `daily`: one file per study session
- `weekly`: one review per week
- `checkpoints`: topic completion reviews

## Suggested naming
- `daily/2026-03-25-linear-algebra.md`
- `weekly/2026-week-13.md`
- `checkpoints/linear-algebra-basics.md`
